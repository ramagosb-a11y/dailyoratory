# Daily Oratory Content Publishing Copyright Checklist

Use this before publishing any new page, reflection, prayer, media asset, PDF, or imported legacy content. This is an operational checklist, not legal advice.

## Authorship And Source

- [ ] Is this content original?
- [ ] If not original, what is the source?
- [ ] Is the creator or author named?
- [ ] Is the source URL saved?
- [ ] Is the source date or access date saved?
- [ ] Is this content adapted from legacy Brotherhood of Ascension material?
- [ ] If legacy, has ownership or permission been confirmed?

## License And Permission

- [ ] Is it public domain?
- [ ] Is it licensed?
- [ ] Is the license URL saved?
- [ ] Is permission documented?
- [ ] Is attribution required?
- [ ] Is the attribution text saved?
- [ ] Is the item safe to publish, or should it remain `Needs Review`?

## Scripture And Church Texts

- [ ] Is this a modern Bible translation?
- [ ] If Scripture text is needed, can Douay-Rheims or another documented public-domain source be used?
- [ ] Can the page use Scripture references and an external link instead of copied text?
- [ ] Is this copied from USCCB, Vatican, a book, a website, or a devotional?
- [ ] Does it include Catechism text? If so, is it short, necessary, and cited by paragraph?
- [ ] Does it include full USCCB daily readings? If yes, do not publish.

## Prayers, Hymns, And Devotions

- [ ] Is this a traditional public-domain prayer?
- [ ] Is the exact prayer source documented?
- [ ] Is this a modern hymn or modern translation?
- [ ] Does it include full modern hymn lyrics? If yes, verify permission or replace.
- [ ] Is a prayer or devotion copied from a modern book, website, or PDF?
- [ ] Should the site link to the source instead of copying?

## Media And Files

- [ ] Are images licensed, original, generated, or public domain?
- [ ] Is creator/source/license metadata saved for each image?
- [ ] Are video embeds from official or clearly authorized channels?
- [ ] Are external PDFs intentionally public and linked rather than copied?
- [ ] Are PowerPoints/PDFs free of copyrighted images and copied text?
- [ ] Do thumbnails, screenshots, or artwork have source and license notes?

## Google Sheets Admin Fields

Before setting `Status = Approved`, confirm these fields are complete where relevant:

- [ ] Creator
- [ ] Author
- [ ] Source Name
- [ ] Source URL
- [ ] License
- [ ] License URL
- [ ] Copyright Status
- [ ] Attribution Text
- [ ] Review Status
- [ ] Reviewed By
- [ ] Reviewed At
- [ ] Notes

## Final Publish Decision

- [ ] Is the text excerpt short enough?
- [ ] Is the attribution visible or available in data?
- [ ] Is the risk level low or accepted by the owner after review?
- [ ] Is this safe to publish?
- [ ] If the item is `High Risk`, `Replace Required`, or `Do Not Publish`, has export been blocked?
