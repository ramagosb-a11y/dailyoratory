# Daily Oratory Content Admin Apps Script

This folder contains the Google Apps Script files for the Daily Oratory Google Sheets content admin system.

Copy these files into a Google Apps Script project attached to a Google Sheet:

- `Code.gs`
- `Config.gs`
- `DataService.gs`
- `Validation.gs`
- `ExportService.gs`
- `JsonBuilder.gs`
- `LogService.gs`
- `Menu.gs`
- `Utilities.gs`

After copying, run `setupDailyOratoryWorkbook` once to create the required tabs, seed `Config`, and apply status dropdowns.

Full setup documentation lives in `docs/google-sheets-content-admin.md`.
